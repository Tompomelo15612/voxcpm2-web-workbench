# Flask 网页端使用说明

## 启动

先进入项目目录并激活已经配置好的 Anaconda 环境：

```bash
cd /path/to/TTS
conda activate voice
python app.py
```

Windows PowerShell 也可以使用 `Set-Location C:\path\to\TTS` 进入项目目录。

浏览器打开：

```text
http://127.0.0.1:5000
```

同一 WiFi / 局域网内的其他电脑访问：

```text
http://你的本机局域网IP:5000
```

例如当前机器可能是：

```text
http://192.168.31.182:5000
```

如果其他电脑打不开，请检查系统防火墙和云平台安全策略是否允许访问 TCP 5000 端口。

## 功能

- 基础克隆：选择参考音频，输入文本后生成。
- 极致克隆：选择参考音频，并填写参考音频对应文本。
- 声音设计：不选择参考音频，直接在文本开头写声音描述。
- 语言 / 方言选择：自动识别、常用语言、中文方言；只用于页面提示和结果标记，不会插入正文。
- 语气模板：自然、冷静克制、剧情独白、愤怒爆发、温柔旁白、悬疑压低。
- 文本规范化：展开数字、日期、金额等内容，适合原始文稿。
- 参考音频降噪：参考音频有底噪时开启，可能轻微改变音色。
- 异常自动重试：生成过短或过长时自动重试，默认开启。
- 长文本自动分段：按句拆分后逐段生成，再拼接成一个 wav。
- 情绪标签快捷插入：向正文插入 `[laughing]`、`[sigh]`、`[Uhm]` 等标签。
- 短剧情绪模板：一键插入冷笑讽刺、压怒质问、失望叹息、震惊反转等常用短剧台词。
- 可控克隆提示词：基础克隆时可加入括号提示词控制情绪、语速和表达方式；极致克隆不使用此项。
- 语速调整：生成后用 `librosa` 做后处理。
- 历史试听：页面会列出 `outputs/` 下的新生成音频，也会显示根目录现有的 `demo_*.wav`。
- 上传参考音频：选择文件后点击“上传并选中”，文件会立即保存到 `references/`，并自动加入、选中参考声音列表。
- 生成进度条：提交后显示当前阶段，长文本分段时会按段推进。
- 静音保护：如果模型生成全静音，会自动清理标签/提示词重试；仍失败则阻止保存并提示。
